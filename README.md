## Setup Guide

Install dependencies:

```bash
pnpm install
```

Then run the app:

```bash
pnpm dev
```

Open http://localhost:3000 to see the voice agent builder.

Or browse the component library in Storybook:

```bash
pnpm storybook
```

Open http://localhost:6006 once it boots.
